Applied Engineering
& Robotics

Design • Build • Program • Compete

Dive into a year-long, hands-on journey to design, build, and program robots! Master the Engineering Design Process (EDP) to create robots using CAD modeling, 3D printing, C++ programming for Arduino, and advanced shop tools like CNC machines, mills, and lathes.

From building autonomous line-following robots to designing FIRST Robotics competition mechanisms, you'll develop real-world engineering skills while exploring career pathways in manufacturing, robotics, and computer science. Safety, teamwork, and ethical design are emphasized throughout.

What You'll Master

Apply the seven-step Engineering Design Process to solve real-world challenges
Create detailed 2D sketches, 3D CAD models, and technical drawings in Fusion 360
Program Arduino microcontrollers using C++ to control motors and sensors
Build and test prototypes, including 3D-printed parts, with data analysis
Fabricate precision parts using CNC routers, mills, lathes, and hand tools
Design and iterate robot mechanisms for functionality or FIRST Robotics
Practice shop safety protocols including PPE, 5S, and lockout/tagout
Build career skills through resume writing and mock interviews
