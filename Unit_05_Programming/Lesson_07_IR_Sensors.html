How IR Reflection Sensors Work

The Physics of Detection

  • IR Emitter: Sends out infrared light (invisible to humans)
  • Surface: Reflects or absorbs the light
  • IR Detector: Measures reflected light intensity
  • Output: HIGH or LOW based on reflection

Reflectivity Basics

White/Light surfaces: Reflect more IR light

Black/Dark surfaces: Absorb more IR light

Digital Output: HIGH or LOW

What is Digital Output?

  • Only two possible states: HIGH (1) or LOW (0)
  • No in-between values (unlike analog)
  • Simple to read and interpret
  • Perfect for line detection (line present or not)
Important: This sensor provides DIGITAL output, not analog. We use digitalRead(), not analogRead().

Sensor Logic (Typical)

Surface Color Reflection Output
White/Light High reflection HIGH (1)
Black/Dark Low reflection LOW (0)

Line Following Setup

For line following, place a black line on a white surface. Sensors return LOW when over the line, HIGH when over the background.

Reading Sensors with digitalRead()

Pin Setup in setup()

All sensor pins must be configured as INPUT before reading.

// IR Sensor pin definitions const int SENSOR_1 = 6; // Far left const int SENSOR_2 = 7; // Left const int SENSOR_3 = 8; // Center const int SENSOR_4 = 9; // Right const int SENSOR_5 = 10; // Far right void setup() { // Configure all sensor pins as inputs pinMode(SENSOR_1, INPUT); pinMode(SENSOR_2, INPUT); pinMode(SENSOR_3, INPUT); pinMode(SENSOR_4, INPUT); pinMode(SENSOR_5, INPUT); // Initialize serial communication for debugging Serial.begin(9600); }

Reading in loop()

Use digitalRead() to get the current state of each sensor.

void loop() { // Read all 5 sensors int s1 = digitalRead(SENSOR_1); int s2 = digitalRead(SENSOR_2); int s3 = digitalRead(SENSOR_3); int s4 = digitalRead(SENSOR_4); int s5 = digitalRead(SENSOR_5); // Now we can use these values... }

Interpreting Sensor Patterns

Understanding Binary Patterns

The combination of 5 sensor readings creates a pattern that tells you where the line is positioned.

Pattern Binary Line Position Action
00100 LOW LOW LOW HIGH HIGH Centered Go straight
01100 LOW HIGH LOW HIGH HIGH Slightly left Turn right slightly
00110 LOW LOW HIGH HIGH HIGH Slightly right Turn left slightly
11000 HIGH HIGH LOW LOW LOW Far left Turn right sharply
00011 LOW LOW LOW HIGH HIGH Far right Turn left sharply
11111 HIGH HIGH HIGH HIGH HIGH No line detected Stop or search
Remember: The exact patterns depend on your line width and sensor spacing. You'll need to test and adjust!

Serial Monitor Debugging

Why Use Serial Monitor?

  • See sensor values in real-time
  • Debug sensor behavior and patterns
  • Verify sensors are working correctly
  • Understand what the robot is "seeing"
// Print individual sensor values void loop() { int s1 = digitalRead(SENSOR_1); int s2 = digitalRead(SENSOR_2); int s3 = digitalRead(SENSOR_3); int s4 = digitalRead(SENSOR_4); int s5 = digitalRead(SENSOR_5); // Print with labels Serial.print("S1:"); Serial.print(s1); Serial.print(" S2:"); Serial.print(s2); Serial.print(" S3:"); Serial.print(s3); Serial.print(" S4:"); Serial.print(s4); Serial.print(" S5:"); Serial.println(s5); // println adds newline delay(100); // Small delay for readability }

Surface Reflectivity Testing

Testing Different Surfaces

  • White paper: Maximum reflection (HIGH)
  • Black tape: Minimum reflection (LOW)
  • Wood/cardboard: Medium reflection (varies)
  • Shiny surfaces: Can cause false readings
Tip: Test your sensors on the actual surface you'll use! Different materials reflect differently.

Sensor Height Matters

The distance between the sensor and surface affects readings:

  • Too close: May get false readings
  • Too far: Weak signal, poor detection
  • Optimal: Usually 2-10mm gap

Environmental Factors

  • Bright ambient light can interfere
  • IR sensors work best indoors
  • Adjust sensitivity if needed

Complete Example: Read All Sensors

// Complete IR Sensor Reading Example // Pin definitions const int SENSOR_1 = 6; const int SENSOR_2 = 7; const int SENSOR_3 = 8; const int SENSOR_4 = 9; const int SENSOR_5 = 10; void setup() { // Set all sensor pins as inputs pinMode(SENSOR_1, INPUT); pinMode(SENSOR_2, INPUT); pinMode(SENSOR_3, INPUT); pinMode(SENSOR_4, INPUT); pinMode(SENSOR_5, INPUT); // Start serial communication Serial.begin(9600); Serial.println("IR Sensor Test Starting..."); } void loop() { // Read all sensors int s1 = digitalRead(SENSOR_1); int s2 = digitalRead(SENSOR_2); int s3 = digitalRead(SENSOR_3); int s4 = digitalRead(SENSOR_4); int s5 = digitalRead(SENSOR_5); // Print pattern Serial.print("Pattern: "); Serial.print(s1); Serial.print(s2); Serial.print(s3); Serial.print(s4); Serial.println(s5); delay(200); // Update 5 times per second }

// Example pattern detection void loop() { int s1 = digitalRead(SENSOR_1); int s2 = digitalRead(SENSOR_2); int s3 = digitalRead(SENSOR_3); int s4 = digitalRead(SENSOR_4); int s5 = digitalRead(SENSOR_5); // Check for center line if (s1 == 1 && s2 == 1 && s3 == 0 && s4 == 1 && s5 == 1) { Serial.println("Line is CENTERED"); } // Add more pattern checks here... else if (s1 == 0) { Serial.println("Line is FAR LEFT"); } else if (s5 == 0) { Serial.println("Line is FAR RIGHT"); } else { Serial.println("No line detected"); } delay(200); }
