Part 1: Base Mount Sketch & Extrusion (21 Points)

STEP 1
Create New Component & Sketch
  • Create a new component named IR_Sensor_Mount
  • Create a new sketch on the XY plane (bottom plane)
  • Set document units to millimeters (mm)
STEP 2
Draw Base Profile

Using the reference dimensions provided, create the U-channel base profile:

Key Dimensions (Sketch 1):

Feature Dimension Notes
Overall Width 84.00 mm Total width of mount
Overall Height 53.00 mm From base to top of flanges
Base Thickness 3.00 mm Bottom platform thickness
Wall Thickness 3.00 mm Side wall thickness
Top Flange Width 16.00 mm Width of mounting flanges
  • Draw the right half of the U-channel profile first
  • Add all dimensional constraints for the right side
  • Use the centerline as your axis of symmetry
  • Use the Mirror command to create the left side geometry
  • The centerline should be positioned at X = 0 (origin)
Verify your sketch is fully constrained (all lines appear BLACK, not blue). The sketch should show 0 degrees of freedom.
STEP 3
Extrude Base Mount (70 Points if Complete)
  • Select the entire U-channel profile
  • Use Extrude command
  • Set distance to 30.00 mm
  • Operation: New Body
  • Direction: Extrude upward (positive Z direction)

Grading Criteria - Part 1:

70 Points - Base mount modeled and extruded correctly, dimensions mostly accurate

75 Points - All of the above + sketch is fully constrained AND all dimensions are labeled

Part 2: Mounting Holes (3-4.5 Points)

STEP 4
Create Sketch on Top Flange
  • Select the top surface of one of the mounting flanges (the horizontal "shelf" at the highest point)
  • Create a new sketch on this plane
  • You'll be drawing circles for mounting holes
STEP 5
Draw Hole Pattern

Create mounting holes for securing the IR sensor board:

Hole Dimensions (Sketch 2):

Feature Dimension Notes
Hole Diameter Ø 6.00 mm Size for M5 hardware clearance
Edge Distance 15.00 mm From hole center to mount edge
Hole Spacing 72.00 mm Center-to-center distance between holes
  • Draw two circles (one on each flange)
  • Position holes with proper edge distance from the ends
  • Use dimensional constraints for all measurements
  • Use geometric constraints (horizontal, coincident, etc.)
Before extruding, verify both circles are fully constrained and positioned correctly on the flanges.
STEP 6
Cut Holes (80 Points if Complete)
  • Select both circles in the sketch
  • Use Extrude command
  • Set operation to Cut
  • Set extent to All (cuts through entire mount)
  • This will create through-holes for mounting hardware

Grading Criteria - Part 2:

80 Points - Mounting holes created and cut through the flanges

85 Points - All of the above + hole sketch is fully constrained AND dimensioned

Part 3: IR Sensor Assembly (2.5 Points)

STEP 7
Insert IR Sensor Component
  • Use Insert → Insert into Current Design
  • Locate and insert the IR sensor PCB assembly file (provided by instructor)
  • The sensor should appear in your workspace as a separate component
STEP 8
Position and Constrain Sensor (90 Points if Complete)

Use joints/constraints to properly mount the sensor to your bracket:

  • Create Rigid Joint between sensor mounting holes and bracket holes
  • Align the sensor PCB so it sits flat on the top flanges
  • The sensor array should be centered within the U-channel
  • Verify the sensor does not intersect with the mount (check for clearance)
  • The sensor pins should face downward into the U-channel cavity

Common Issue Alert!

At this stage, you may notice the sensor does NOT fit properly. The holes might not align, or there's interference. This is intentional! You'll need to identify and fix the dimensional error in Part 4.

Grading Criteria - Part 3:

90 Points - IR sensor inserted and constrained to the mount (even if fit is imperfect)

Part 4: Design Correction (100 Points - Perfect Score!)

STEP 9
Identify the Problem

If the sensor doesn't fit properly, there's likely a dimensional error in your hole spacing or edge distance.

Common issues to check:

  • Are the mounting holes aligned with the sensor's holes?
  • Is the hole spacing (center-to-center distance) correct?
  • Are the holes too close or too far from the edges?
  • Does the sensor PCB width match your mount's flange spacing?
STEP 10
Correct the Dimension (3 Points!)

The secret to achieving full points (30/30):

💡 Design Iteration Hint

  • Measure the actual hole spacing on the IR sensor PCB
  • Edit Sketch 2 (the hole pattern sketch)
  • Update ONE dimension that's preventing proper fit
  • Re-constrain if needed to maintain full constraints
  • The corrected dimension should allow perfect alignment

Hint: The actual sensor hole spacing is likely NOT 72.00 mm. Use the Inspect → Measure tool to find the true distance!

Once corrected, the sensor should fit perfectly with mounting holes aligned and no interference!

Grading Criteria - Part 4:

3 Points - Dimensional error identified and corrected. Sensor fits perfectly with proper constraints.

Tips for Success

🎯 Constraint Strategies

  • Start with the origin: Anchor one point to the origin or a construction line
  • Use construction geometry: Centerlines make mirroring much easier
  • Dimension everything: Don't rely on "eyeballing" - add explicit dimensions
  • Check your work: Zoom in and verify dimensions match the specification

🔧 Common Mistakes to Avoid

  • Under-constrained sketches: Blue lines indicate missing constraints - add them!
  • Wrong extrusion direction: Watch the preview arrow - make sure it goes the right way
  • Wrong plane selection: Double-check you're sketching on the correct surface
  • Forgetting to measure: Use the Measure tool to verify sensor dimensions
  • Not testing fit: Always check assembly fit before assuming it's correct

⚡ Time Management

  • Part 1 (Base): 20-25 minutes - Take your time with constraints
  • Part 2 (Holes): 15-20 minutes - Precision matters here
  • Part 3 (Assembly): 10-15 minutes - Joint placement is key
  • Part 4 (Correction): 10-15 minutes - Measure, edit, verify
