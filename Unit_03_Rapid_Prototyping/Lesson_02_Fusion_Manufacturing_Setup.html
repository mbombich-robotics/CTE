🔄 Design vs. Manufacturing Workspace

What's the Difference?

DESIGN Workspace

  • Purpose: Create 3D models
  • Tools: Sketch, Extrude, Fillet
  • Output: .f3d, .step, .stl files
  • Focus: Geometry & dimensions

MANUFACTURE Workspace

  • Purpose: Make real parts
  • Tools: Toolpaths, Operations
  • Output: G-code (.nc, .tap, .gcode)
  • Focus: How to cut material

The Manufacturing Workflow

1 Design part in Design workspace

2 Switch to Manufacture workspace

3 Select machine & tool

4 Create toolpaths (how tool cuts material)

5 Simulate cutting process

6 Generate G-code with post-processor

7 Load G-code into CNC machine

🏭 Machine Libraries Explained

What is a Machine Library?

A machine library is a digital definition of your CNC machine. It tells Fusion 360:

  • How big the cutting area is (work envelope)
  • What axes the machine has (X, Y, Z, rotational)
  • Maximum speeds and accelerations
  • Safety limits and machine capabilities

Why We Need Machine Libraries

  • Collision prevention: Fusion won't let you design toolpaths outside the machine's range
  • Realistic simulation: Preview exactly how your part will be cut
  • Optimized G-code: Code is tailored to your specific machine
  • Setup sheets: Generates instructions for machine operators

⚠️ Important: Using the wrong machine library can generate G-code that damages equipment or produces incorrect parts. Always verify your machine selection!

📝 What is a Post-Processor?

The Translation Layer

Fusion 360 creates toolpaths (the path the cutting tool follows). But CNC machines can't read toolpaths directly—they need G-code.

A post-processor translates Fusion's toolpaths into machine-specific G-code.

Why Different Post-Processors?

  • Different controllers use different G-code dialects (GRBL vs. LinuxCNC vs. Mach3)
  • Machines have different capabilities (e.g., plasma needs torch height control codes)
  • File format preferences (.nc vs .tap vs .gcode)
  • Custom safety protocols and startup sequences

Our Post-Processors

  • omio-grbl.cps → For OMIO CNC Router (GRBL controller)
  • FireControl-v1.6.cps → For Crossfire Pro Plasma (FireControl software)

⚙️ Installing Post-Processors

Locating the Post-Processor Folder

Windows Path:
C:\Users\YourName\AppData\Roaming\Autodesk\
  Fusion 360 CAM\Posts

Mac Path:
~/Library/Application Support/Autodesk/
  Fusion 360 CAM/Posts

Installation Steps

  • 1 Download omio-grbl.cps and FireControl-v1.6.cps
  • 2 Press Windows Key + R → Type %appdata% → Press Enter
  • 3 Navigate to Autodesk → Fusion 360 CAM → Posts
  • 4 Copy both .cps files into this folder
  • 5 Restart Fusion 360

⚠️ Troubleshooting: If you don't see the "Posts" folder, create it manually. The path must be exact—spaces and capitalization matter!

🔨 Tool Libraries Explained

What is a Tool Library?

A tool library contains definitions of every cutting tool you own:

  • Physical dimensions (diameter, length, flutes)
  • Material properties (HSS, carbide, diamond)
  • Cutting parameters (speeds, feeds, depths)
  • Tool number (for automatic tool changers)

Why Accurate Tool Data Matters

  • Collision prevention: Fusion simulates with actual tool geometry
  • Optimal cutting: Wrong speeds = broken tools or burned material
  • Finish quality: Feed rates affect surface smoothness
  • Tool life: Proper parameters prevent premature wear

Professional Shops: Tool libraries can have 100+ tools. We'll start with 2 essential tools for robot manufacturing.

🛠️ Creating Tool Definitions

Tool Type:         Plasma (consumable)

Pierce Height:     3.8 mm
Cut Height:        1.5 mm
Pierce Delay:      0.5 seconds
Lead-in Distance:  3 mm

Cutting Parameters:
  Feed Rate:       4000 mm/min (mild steel)
  Max Thickness:   12.7 mm (1/2")

➕ Adding Tools to Your Library

Creating the End Mill

  • 1 Manufacture workspace → Manage → Tool Library
  • 2 Click New Tool → Select Flat End Mill
  • 3 Enter tool geometry (diameter, length, flutes)
  • 4 Switch to Cutting Data tab
  • 5 Enter speeds/feeds for each material (plastic, wood, aluminum)
  • 6 Click Accept to save

💡 Pro Tip: Material-Specific Settings

The same tool cuts different materials at different speeds:

  • Polycarbonate: 18,000 RPM @ 1500 mm/min
  • Plywood: 18,000 RPM @ 2000 mm/min
  • Aluminum: 12,000 RPM @ 800 mm/min (use coolant!)

⚠️ Safety Note: Never exceed manufacturer's recommended RPM for your tool diameter. Formula: Surface Speed (SFM) = (RPM × Diameter × π) / 12
