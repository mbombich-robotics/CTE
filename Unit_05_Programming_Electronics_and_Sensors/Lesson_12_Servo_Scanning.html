How Servo Motors Work

Servo sweeping 0° to 180°

PWM Signal (pulse width varies)

Inside a Servo

  • DC Motor: Provides rotation
  • Gearbox: Increases torque, reduces speed
  • Potentiometer: Senses current position
  • Control Circuit: Compares desired vs actual position

Key Specs: SG90 Servo

PWM Position Control

Pulse Width = Position

The servo reads the width of the HIGH pulse to determine where to move

─────────────────────────────────────
Minimum      1.0 ms          0°
Center       1.5 ms          90°
Maximum      2.0 ms          180°

Wiring the Servo

   SG90 Servo                 Arduino Uno
  ┌─────────┐                ┌─────────────┐
  │  Brown  │───────────────→│  GND        │
  │  Red    │───────────────→│  5V         │
  │  Orange │───────────────→│  Pin 9      │
  └─────────┘                └─────────────┘

Wire Colors (Standard):
  • Brown/Black = Ground (GND)
  • Red = Power (5V)
  • Orange/Yellow/White = Signal (PWM)

Power via USB

For a single scanning servo, the Uno's 5V pin powered by computer USB is sufficient. The SG90 draws ~100–250mA sweeping freely — well within the ~420mA remaining after the Uno itself.

Avoid stalling the servo — hitting a physical stop and holding can spike current and cause a brownout.

Pin Choice

The Servo library handles PWM timing internally. Pin 9 is a good choice — it's PWM-capable and leaves other common pins free.

If you ever add drive motors, move servo power to an external 5V supply with a shared ground.

Using the Servo Library

#include <Servo.h> Servo myServo; // Create servo object void setup() { myServo.attach(9); // Attach servo to pin 9 } void loop() { myServo.write(0); // Move to 0 degrees delay(1000); myServo.write(90); // Move to 90 degrees (center) delay(1000); myServo.write(180); // Move to 180 degrees delay(1000); }

Key Functions

  • attach(pin) - Connect servo to a pin
  • write(angle) - Move to angle (0-180)
  • read() - Get current angle
  • detach() - Release the servo

Note

The servo moves as fast as it can to the target position. For smooth motion, we need to move in small increments with delays.

Smooth Sweeping Motion

#include <Servo.h> Servo scanServo; const int servoPin = 4; void setup() { scanServo.attach(servoPin); scanServo.write(90); // Start at center delay(500); } void loop() { // Sweep from 0 to 180 for (int angle = 0; angle <= 180; angle += 1) { scanServo.write(angle); delay(15); // Wait for servo to reach position } // Sweep from 180 back to 0 for (int angle = 180; angle >= 0; angle -= 1) { scanServo.write(angle); delay(15); } }

Understanding the Sweep

  • angle += 1: Move 1 degree at a time for smooth motion
  • delay(15): Give servo time to reach each position
  • Full sweep takes: 181 positions × 15ms = ~2.7 seconds each direction
  • Adjust delay for faster/slower sweep (try 10-30ms)

Building the Scanner Assembly

Components Needed

  • SG90 or MG90S servo motor
  • HC-SR04 ultrasonic sensor
  • Servo horn (cross or single arm)
  • Mounting bracket or 3D printed holder
  • Small screws or hot glue
  • Jumper wires

Mount Tips

  • Keep sensor centered over servo axis
  • Secure all wires away from moving parts
  • Test range of motion before final mounting

Combining Servo + Ultrasonic

#include <Servo.h> // Pin definitions const int servoPin = 4; const int trigPin = 2; const int echoPin = 3; Servo scanServo; void setup() { Serial.begin(115200); scanServo.attach(servoPin); pinMode(trigPin, OUTPUT); pinMode(echoPin, INPUT); scanServo.write(90); delay(500); } void loop() { // Sweep and scan for (int angle = 0; angle <= 180; angle += 10) { scanServo.write(angle); delay(50); // Wait for servo long distance = getDistance(); Serial.print("Angle: "); Serial.print(angle); Serial.print("° Distance: "); Serial.print(distance); Serial.println(" cm"); } delay(500); } long getDistance() { digitalWrite(trigPin, LOW); delayMicroseconds(2); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW); long duration = pulseIn(echoPin, HIGH, 30000); return duration / 58; }

Lab Exercise: Build & Test

Today's Goals

  1. Wire up the servo motor and test basic movement
  2. Mount the ultrasonic sensor on the servo
  3. Upload the combined scan code
  4. Verify readings at different angles
  5. Adjust sweep speed and step size as needed

Serial Monitor Calibration (115200 baud)

The sketch starts in command mode — type an angle and press Enter to move the servo directly. Use this to find your servo's real endpoints before mounting.

  • Type 0 → move to 0°  |  Type 90 → center  |  Type 180 → move to 180°
  • Type scan → start continuous sweep mode
  • Type any angle while scanning → stop sweep and hold that position

Test Checklist

  • ☐ Serial Monitor opens at 115200 baud
  • ☐ Type angles to verify full range of motion
  • ☐ Type "scan" — servo sweeps smoothly 0° to 180°
  • ☐ No binding or scraping during sweep
  • ☐ Sensor readings appear after type "scan"
  • ☐ Wires don't tangle during rotation

Troubleshooting

  • Servo jitters: Check power supply
  • No movement: Verify signal wire on pin 9
  • Limited range: Check for obstructions
  • Bad readings: Sensor may be loose
  • Serial not responding: Confirm 115200 baud
