Essential Code Snippets

Ultrasonic Distance:
long duration = pulseIn(echoPin, HIGH);
long distance = duration * 0.034 / 2; // cm
Servo Control:
myServo.attach(5); // Setup
myServo.write(90); // Move to 90°
Claw Functions:
void openClaw() { clawServo.write(OPEN_POS); }
void closeClaw() { clawServo.write(CLOSED_POS); }
void gripPercent(int pct) { clawServo.write(map(pct, 0, 100, OPEN_POS, CLOSED_POS)); }

Common Fixes

  • Servo won't move: Check 5V power, verify pin
  • Distance = 0: TRIG/ECHO swapped, check wiring
  • Erratic behavior: Low battery, loose wires
  • Won't upload: Check COM port, press reset
  • Serial garbage: Match baud rate (115200)
