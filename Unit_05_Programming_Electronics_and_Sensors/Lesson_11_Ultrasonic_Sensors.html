How Ultrasonic Sensors Work

The Physics

  • Transmitter: Sends ultrasonic pulse (40kHz)
  • Sound travels: ~343 m/s in air
  • Pulse bounces: Off objects and returns
  • Receiver: Detects the echo
  • Timer: Measures round-trip time

Distance Formula

Distance = Time × Speed ÷ 2

We divide by 2 because sound travels TO the object AND back

The 4 Pins

  • VCC: 5V power supply
  • GND: Ground connection
  • TRIG: Trigger input (start measurement)
  • ECHO: Echo output (pulse width = time)

Important: The ECHO pin outputs 5V. Some boards need a voltage divider!

Wiring the HC-SR04

Wire Connections

Arduino Pin → Sensor Pin
5V→VCC
GND→GND
Pin 9→TRIG
Pin 10→ECHO

Pin Choices

  • TRIG can be any digital output pin
  • ECHO can be any digital input pin
  • We'll use pins 9 and 10 in our examples

Wiring Tips

  • Double-check VCC and GND - reversed power can damage the sensor
  • Use short, solid jumper wires for reliable connections

How the Timing Works

TRIG Pin:
         ┌────┐
─────────┘    └─────────────────────────────────────────
         │10µs│

ECHO Pin:
                   ┌─────────────────────┐
───────────────────┘                     └───────────────
                   │←── pulse width ────→│
                   │   (proportional     │
                   │    to distance)     │

Timeline:
1. Arduino sends 10µs HIGH pulse on TRIG
2. Sensor sends 8 ultrasonic pulses at 40kHz
3. ECHO goes HIGH
4. Sound bounces off object
5. ECHO goes LOW when echo received
6. Pulse width = round-trip time

pulseIn() function measures how long ECHO stays HIGH in microseconds

Converting Time to Distance

Speed of Sound

343 meters/second = 0.0343 cm/µs

The Calculation

1. Get duration in microseconds

duration = pulseIn(echoPin, HIGH)

2. Calculate distance

distance = duration × 0.0343 / 2

3. Simplified formula:

distance = duration / 58

Example

If pulseIn() returns 1160 µs:

  • 1160 × 0.0343 = 39.8 cm (round trip)
  • 39.8 ÷ 2 = 19.9 cm
  • Or: 1160 ÷ 58 = 20 cm

Basic Distance Reading Code

// HC-SR04 Ultrasonic Sensor - Basic Distance Reading const int trigPin = 9; const int echoPin = 10; void setup() { Serial.begin(115200); pinMode(trigPin, OUTPUT); pinMode(echoPin, INPUT); Serial.println("Ultrasonic Sensor Ready!"); } void loop() { long distance = getDistance(); Serial.print("Distance: "); Serial.print(distance); Serial.println(" cm"); delay(100); } long getDistance() { // Send trigger pulse digitalWrite(trigPin, LOW); delayMicroseconds(2); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW); // Read echo pulse duration long duration = pulseIn(echoPin, HIGH); // Convert to distance in cm long distance = duration / 58; return distance; }

Understanding the Code

The Trigger Sequence

digitalWrite(trigPin, LOW); delayMicroseconds(2); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW);
  • Start LOW to ensure clean pulse
  • Go HIGH for exactly 10µs
  • Return to LOW to complete trigger

Reading the Echo

long duration = pulseIn(echoPin, HIGH);

pulseIn(pin, HIGH) waits for the pin to go HIGH, then times how long it stays HIGH before going LOW again.

Returns the pulse width in microseconds.

Handling Edge Cases

What Can Go Wrong?

  • Out of range: Object too far (>400cm) or too close (<2cm)
  • No echo: Sound absorbed or deflected away
  • Timeout: pulseIn() waits forever by default
long getDistanceSafe() { // Send trigger pulse digitalWrite(trigPin, LOW); delayMicroseconds(2); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW); // Read echo with timeout (30ms = ~500cm max) long duration = pulseIn(echoPin, HIGH, 30000); // Check for timeout (returns 0) if (duration == 0) { return -1; // Error: no echo received } long distance = duration / 58; // Validate range if (distance < 2 || distance > 400) { return -1; // Out of valid range } return distance; }
