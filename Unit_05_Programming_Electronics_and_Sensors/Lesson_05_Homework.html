Assignment: Potentiometer Brightness Control

Build a circuit where a potentiometer controls LED brightness using analog input and PWM output.

Core Requirements

  • Connect potentiometer to A0
  • Connect LED to PWM pin 9
  • Use analogRead() to read potentiometer value
  • Use map() function to convert 0-1023 to 0-255
  • Use analogWrite() to set LED brightness
  • Print both values to Serial Monitor

Expected behavior: When you turn the potentiometer, the LED should smoothly change brightness from off to full brightness.
