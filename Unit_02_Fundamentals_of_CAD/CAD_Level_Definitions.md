# CAD Unit — Level Definitions (All 6 Components)

*Level applies per component. Students choose their level — grade reflects what they complete.*
*L1 alone → C | L1+L2 → B | L1+L2+L3 → A | L4 → still A (challenge only)*

---

## Component 1 — Motor Mount (3D print, PLA)
**Functional requirements:** Correct shaft alignment, correct bolt pattern, adequate wall thickness to survive assembly.

| Level | Description |
|---|---|
| **L1** | Follow the instructional video; replicate Mr. B's motor mount exactly. Print and test-fit. |
| **L2** | Modify the design: add gussets for rigidity, adjust wall thickness, improve bolt access, or change the retention method. Must still pass test-fit. |
| **L3** | Completely unique design that meets all functional requirements: correct shaft centerline, correct bolt pattern, no interference with wheel or frame. |
| **L4** | Parametric design — use Fusion 360 parameters so the mount adapts to a different motor diameter or bolt pattern by changing one value. Document the parameters. |

---

## Component 2 — IR Sensor Mount (3D print, PLA)
**Functional requirements:** Positions TCRT5000 sensor at exactly 15mm ground clearance, correct forward orientation, secure attachment to deck.

| Level | Description |
|---|---|
| **L1** | Follow the instructional video; replicate Mr. B's IR sensor mount. Print, install, and verify 15mm clearance with a caliper. |
| **L2** | Modify: add a cable management channel, improve the attachment method, add adjustability (slotted holes), or reduce print material without losing rigidity. |
| **L3** | Unique design that maintains 15mm clearance and correct sensor orientation; may attach differently, add a shield, or serve two sensors in one mount. |
| **L4** | Design an integrated dual IR sensor mount (left + right sensor in one part) with independent adjustability for each sensor. |

---

## Component 3 — Battery Holder (3D print, PLA)
**Functional requirements:** Securely retains the 12V LiPo battery, includes wire relief cutouts so wires exit cleanly without sharp bends.

| Level | Description |
|---|---|
| **L1** | Follow the instructional video; replicate Mr. B's battery holder with wire relief cutouts. Print and test-fit with battery. |
| **L2** | Modify: improve retention method (clip, strap slot, locking tab), improve wire exit routing, or add a strain relief feature. |
| **L3** | Unique design with a different retention approach (snap-in, sliding rail, Velcro-integrated slot) that still secures the battery and routes wires cleanly. |
| **L4** | Design for a different battery form factor OR integrate a fuse holder into the battery holder as a single printed part. |

---

## Component 4 — Ultrasonic Sensor Mount (3D print, PLA)
**Functional requirements:** Positions HC-SR04 sensor facing directly forward, secure fit, wires exit without blocking the sensor face.

| Level | Description |
|---|---|
| **L1** | Follow the instructional video; replicate Mr. B's ultrasonic mount. Print, install, and verify sensor faces forward with no wire obstruction. |
| **L2** | Modify: add a tilt adjustment (slotted pivot), improve wire management, reduce weight, or improve the attachment point. |
| **L3** | Unique design that correctly positions the sensor; may use a different attachment strategy or integrate with another component. |
| **L4** | Design an integrated pan/tilt bracket that mounts the HC-SR04 on a servo for the scanner project — combines C4 with scanner hardware in one designed part. |

---

## Component 5 — Robot Deck (CNC cut, polycarbonate)
**Functional requirements:** Correct hole patterns for motors, caster, and component mounts; wire pass-throughs in useful locations; fits within robot size envelope.

| Level | Description |
|---|---|
| **L1** | Follow the instructional video; replicate Mr. B's deck layout. Verify all hole positions with calipers before submitting for CNC cut. |
| **L2** | Modify the layout: relocate components for better weight distribution, add wire pass-throughs, improve access for maintenance, or adjust hole sizing for better fit. All functional holes must still be correct. |
| **L3** | Unique deck layout within the size envelope: different component arrangement optimized for your robot's specific build. Must pass all functional checks before cut. |
| **L4** | Parametric deck — use Fusion 360 parameters for wheelbase, motor spacing, and component positions so the deck can be regenerated for a different robot configuration by changing inputs. Document the parameters. |

---

## Component 6 — Electronics Mount (CNC cut, polycarbonate)
**Functional requirements:** Correct standoff spacing for carrier board mounting hole pattern; correct spacing for 2× VNH5019 modules; fits on deck without interference.

| Level | Description |
|---|---|
| **L1** | Follow the instructional video; replicate Mr. B's electronics mount. Verify standoff positions match the carrier board hole pattern with calipers before submitting for CNC cut. |
| **L2** | Modify: add cable management features (tie-down slots, wire channels), adjust standoff heights for better access, or add mounting for an additional component (switch, Wago block). |
| **L3** | Unique layout that correctly spaces all electronics per spec; may arrange carrier board and VNH5019 modules differently or integrate additional mounts in a single part. |
| **L4** | Modular electronics mount — design it so the carrier board and VNH5019 modules can be swapped or repositioned without cutting a new deck. May use a rail system or adjustable standoff slots. |

---

## Component 7 — Rear Wheel Hub (3D print, PLA)
**Functional requirements:** 3/8" hex bore fits motor shaft without slop; mates with tread (C8); withstands drive torque without cracking.

| Level | Description |
|---|---|
| **L1** | Follow the instructional video; replicate Mr. B's wheel hub. Print and test-fit on motor shaft. |
| **L2** | Modify: improve retention (set screw boss, press-fit feature), reduce weight with cutouts, or adjust spoke geometry. |
| **L3** | Unique hub design that maintains 3/8" hex bore and correct interface with C8 tread; may use a different tread attachment method. |
| **L4** | Parametric hub — use Fusion 360 parameters for hex bore size and tread diameter so the hub adapts to a different motor shaft or tread size. |

---

## Component 8 — Rear Wheel Tread (3D print, TPU)
**Functional requirements:** 3" outer diameter; flexible enough to grip surface; bonds or snaps onto hub (C7) without slipping under load; consistent diameter for straight tracking.

| Level | Description |
|---|---|
| **L1** | Follow the instructional video; replicate Mr. B's tread profile. Print in TPU and test-fit on C7 hub. Verify no slippage under hand torque. |
| **L2** | Modify: change tread pattern (grooves, knobs) for different grip characteristics; adjust wall thickness; improve hub retention method. |
| **L3** | Unique tread design with a different profile or attachment strategy; must still maintain consistent OD and grip the hub securely. |
| **L4** | Tread pattern optimized through iteration — print 3+ variants with different patterns, test on the line following track, document grip vs. speed tradeoff. |

---

## Notes for 2nd-Year Students

Students who have prior CAD experience will complete L1 faster. When ahead:
1. Move immediately to L2 or L3 on the component they just finished.
2. If all 6 components are at L1 and time remains, pull from the **Continuous Improvement List** for additional design challenges.

L4 challenges are designed for students who have completed L3 and want to push further — there is no additional grade benefit beyond A.

## Grading Summary

| Completed levels | Grade outcome |
|---|---|
| L1 on all 8 components | C |
| L1 + L2 on all 8 components | B |
| L1 + L2 + L3 on all 8 components | A |
| Any L4 work | A (no change — challenge only) |

*Partial completion (e.g., L2 on 4 components, L1 on 2) earns a grade between C and B at teacher discretion.*
